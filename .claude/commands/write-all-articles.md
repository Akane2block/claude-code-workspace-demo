# write-all-articles

## やること

`記事/2_materials/` にある全素材を順番に処理して、記事を一括生成する。

## 手順

1. `記事/2_materials/` のファイル一覧を取得する
2. `記事/3_output/` にすでに対応する記事がない素材だけを対象にする
3. 各素材に対して以下を繰り返す：
   a. 素材を読む
   b. `記事/1_prompts/` から適切なプロンプトを選ぶ
   c. 記事を生成する
   d. `記事/3_output/YYYYMMDD-タイトル.md` として保存する
4. 全件完了したら処理結果の一覧を報告する

## 完了後の報告フォーマット

|------------|---------------|------------|
| 01_〇〇.md | 01_AI活用事例 | 20260311-〇〇.md |
| ...        | ...           | ...        |
