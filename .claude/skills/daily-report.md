# 日報作成スキル

`仕事/日報/` フォルダの作業ログを読んで、今日の日報を自動生成するスキルです。

## 手順

1. `仕事/日報/日報テンプレート.md` を読んでフォーマットを確認する
2. 今日の日付（YYYY-MM-DD）を取得する
3. テンプレートに従って日報を作成する
4. `仕事/日報/output/YYYY-MM-DD_日報.md` として保存する
5. 「日報を作成しました：ファイルパス」と報告する

## 補足

- 参照できる作業ログがない場合は「今日の作業内容を教えてください」と聞く
- 箇条書きで簡潔にまとめる
- 明日の予定欄は空欄のままでOK（後でユーザーが記入）
