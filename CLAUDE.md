# このワークスペースのルール

Claude Code がこのフォルダを開いたときに自動で読み込まれる設定ファイルです。

## フォルダの使い方

| フォルダ | 用途 |
|---------|------|
| `日記/` | 日々の記録・メモ |
| `仕事/日報/` | 日報の管理 |
| `仕事/議事録/` | MTG・打ち合わせの記録 |
| `パーソナル/` | 個人メモ・タスク管理 |
| `記事/1_prompts/` | 記事の書き方テンプレート |
| `記事/2_materials/` | 記事の素材（文字起こし・メモ） |
| `記事/3_output/` | 生成した記事の保存先 |

## ファイル保存のルール

- 日報は `仕事/日報/output/YYYY-MM-DD_日報.md` に保存する
- 議事録は `仕事/議事録/YYYY-MM-DD_会議名.md` に保存する
- 日記は `日記/YYYY-MM-DD.md` に保存する

## 出力スタイル

- ファイル名には必ず日付を入れる（例: `2026-03-11_日報.md`）
- 長い文章よりも箇条書きを優先する
- 確認なしにファイルを削除しない

## 記事生成スキル

| コマンド | 説明 |
|---------|------|
| `/write-article` | 素材1件から記事を生成 |
| `/write-all-articles` | 全素材を一括処理して記事を量産 |
